## Getting Started

### Setup

Open a terminal and run the following commands

1. Clone the repo
    ```sh
    git clone --recursive https://github.com/HugoKempe/MyReactWebsite
    ```
1. Change directory
    ```sh
    cd MyReactWebsite/my-react-app
    ```
2. Install NPM packages
    ```sh
    npm install
    ```
3. Start the server
    ```
    npm run dev
    ```
